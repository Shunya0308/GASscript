# GASscript

## 概要

## ① task_notify.gs
このスクリプトは、TrelloとLINEを連携し、毎日定刻に現在残っているタスクを出力してくれるスクリプトになります。


### API
・LINE Messaging API
・Trello API

## ② calendar_notify.gs
このスクリプトは、GoogleカレンダーとLINEを連携して、予定の追加、定期的な予定通知、今日の予定、一週間の予定をLINE上で出力してくれるスクリプトになります。


### API
・LINE Messaging API
・Google Calendar API
